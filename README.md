# GraphQL Profile

A personal profile dashboard built by **Abrar Naser** using **HTML, CSS, JavaScript, and GraphQL**.

This project displays my profile data in a clean dashboard layout, including:
- User information
- XP statistics
- Level
- Audit ratio
- XP by project chart
- XP timeline
- Top skills chart

## Live Demo
[View Project](https://graphql-abrnaser.netlify.app/)

## About
I created this project to practice working with **GraphQL APIs**, handling user data, and presenting it in a visually structured way through a responsive and modern interface.
